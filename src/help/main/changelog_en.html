Changelog


efa 2.0.1 (Release Date 15.04.2012)

  • Waters can be entered for tours to unknown destinations, and destination infos can optionally be displayed when entering a session
  • Improved Statistics functionalities
  • Additional fields in boat administration for boat repair costs and related data
  • Default boat type for boats with multiple boat types
  • Bugfixes

efa 2.0.0 (Release Date 26.02.2012)

  • New Data Model
  • New Administration Interface
  • Remote Access through GUI and CLI (experimental)
  • Impoved Data Safety through Crash Recovery, Mirror Directory, Backup and Restore
  • Adaptations for Canoeing
  • Internationalization and Translation into German, English, French, Italian, Greek, Danish, and Dutch
  • Numerour Optimizations for Use in a Boathouse
  • Many small Improvements

Previous versions of efa have only been released in German. See the German Changelog for a list of changes in previous versions.
